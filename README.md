# Kanban Board Project

## Features
✔️ User authentication  
✔️ Board & Column management  
✔️ Task assignment with status & due date  
✔️ Drag & drop tasks  
✔️ Board member invitations  

## Tech Stack
- Node.js + Express (Backend)
- React (Frontend)
- MongoDB (Database)
